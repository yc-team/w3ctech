nodeVersionMg
=============

node version manager


#### 很多人有没有想过node如何管理版本？

目前推荐的就两个工具：

1. [nvm](https://github.com/creationix/nvm)
2. [n](https://github.com/visionmedia/n) 

> 很多人在npm官网下的那个0.0.3的 [nvm](https://www.npmjs.org/package/nvm) 其实不是我上面的那个
